Player Classes

Classes define the main attribute and base skills for your character. Skills can also be obtained from in game items, trainers and quests. A classes main attribute has a reduced training cost and factors into various game checks.

Name Main Stat About
Madbomb Constitution Uses their own bodies for explosive assaults. Primary weapon: Panzerhands
Hacker Intelligence Uses code and exploits to confuse and disable enemies. Primary weapon: Decks
Origamist Dexterity Creates paper objects to set up big-damage combos. Primary weapon: Guns
